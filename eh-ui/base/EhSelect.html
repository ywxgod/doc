# EhSelect

# 继承

无

# 描述

封装了element-ui的Select组件，原Select组件的属性，方法，事件都可用，另外还增加了valueField, labelField等属性。 原Select组件糅合了太多的功能，EhSelect试图区分和抽象一些基本的功能，然后再分别扩展。对于使用者来说功能会更加直接。

# 属性

属性
labelField：String = 'label'
指定数据源(options)中用来获取label的属性，即指定读取哪个属性来显示在options列表中。
valueField：String = 'id'
指定数据源(options)中用来获取value的属性，即指定读取哪个属性来指示显示项的值
options：Array
指定数据源(options)中用来获取value的属性，即指定读取哪个属性来指示显示项的值
group：Boolean
是否分组显示下拉菜单数据

# 方法

无

# 事件

事件
action：String
调用EhItemRenderer的dispatch方法时默认会触发action事件

# 属性详情

labelField

labelField的值必须是options数组中元素的某个属性名称，其默认值为label。如options为：

[ 
    { city: '广州', province: '广东', id: 1 },
    { city: '武汉', province: '湖北', id: 2 }
]
1
2
3
4

我们可以指定labelField的值为city，这样EhSelect下拉菜单将会显示城市名称：广州-武汉；

如果labelField的值为province，下拉菜单则显示：广东-湖北

valueField

valueField用法同labelField，默认值为id。用于options选项中选中的值。

options

options指定EhSelect下拉菜单的数据源。options数组元素中我们可以指定itemRenderer属性，用于渲染自定义下拉菜单项。如：

 [
    { id: 1, name: '宋江', age: 43, itemRenderer: EhSelectItemRendererSample },
    { id: 2, name: '吴用', age: 40, itemRenderer: EhSelectItemRendererSample },
    { id: 3, name: '卢俊义', age: 34, itemRenderer: EhSelectItemRendererSample },
    { id: 4, name: '武松', age: 30, itemRenderer: EhSelectItemRendererSample }
]

itemRender可以继承自EhItemRenderer。请参考EhItemRenderer使用说明。

group

group默认值为false，即不分组显示。如果group为true，需要options支持，options必须符合以下结构：具体用法可以参考原Select组件

[ 
{ 
    label: '热门城市',
    options: [{
        value: 'Shanghai',
        label: '上海'
    }, {
        value: 'Beijing',
        label: '北京'
    }]
}, 
{
    label: '城市名',
    options: [{
        value: 'Chengdu',
        label: '成都'
    }, {
        value: 'Shenzhen',
        label: '深圳'
    }, {
        value: 'Guangzhou',
        label: '广州'
    }, {
        value: 'Dalian',
        label: '大连'
    }]
}
]

# 事件详情

action

当自定义itemRenderer时，我们的itemRenderer须继承自EhItemRenderer。

在EhItemRenderer中定义了item属性、dispatch方法。

item指向当前被渲染的数据项，即options数组的某个元素；

dispatch方法会调用EhItemRenderer的$emit方法带上item和事件对象向外发送事件action事件，此事件会被EhSelect以相同事件名称转发出来，所以EhSelect组件会派发action事件

参数名称 说明
数组：[item, ...rest] item为当前itemRenderer所渲染的options项，来自于options数组。 rest为自定义参数数组，在自己实现的itemRenderer中往往需要带上额外的参数

EhSelect其他事件，比如change等，可以参考原Select组件

EhComboBoxSample.vue文件代码：

<template>
    <div>
        <eh-select
                labelField="name"
                @action="onAction" @change="onChange"
                :options="options1" v-model="userId1">
        </eh-select>
        <eh-select
                labelField="userName"
                valueField="userId"
                @change="onChange"
                :options="options2" v-model="userId2">
        </eh-select>
        <eh-select
                v-model="city"
                value-field="value"
                :options="groupOptions"
                group="true">
        </eh-select>
    </div>
</template>

<script>
    import EhSelectItemRendererSample from "./EhSelectItemRendererSample";

    import enui from 'ele-enhance-ui';
    const { EhSelect } = enui;

    export default {
        name: 'EhSelectSample',
        components: { EhSelect },
        data() {
            return {
                options1: [
                    { id: 1, name: '宋江', age: 43, itemRenderer: EhSelectItemRendererSample },
                    { id: 2, name: '吴用', age: 40, itemRenderer: EhSelectItemRendererSample },
                    { id: 3, name: '卢俊义', age: 34, itemRenderer: EhSelectItemRendererSample },
                    { id: 4, name: '武松', age: 30, itemRenderer: EhSelectItemRendererSample }
                ],
                options2: [
                    { userId: 1, userName: '风清扬' },
                    { userId: 2, userName: '镇远和尚' },
                    { userId: 3, userName: '任我行' },
                    { userId: 4, userName: '东方不败' }
                ],
                groupOptions: [
                    {
                        label: '热门城市',
                        options: [{
                            value: 'Shanghai',
                            label: '上海'
                        }, {
                            value: 'Beijing',
                            label: '北京'
                        }]
                    },
                    {
                        label: '城市名',
                        options: [{
                            value: 'Chengdu',
                            label: '成都'
                        }, {
                            value: 'Shenzhen',
                            label: '深圳'
                        }, {
                            value: 'Guangzhou',
                            label: '广州'
                        }, {
                            value: 'Dalian',
                            label: '大连'
                        }]
                    }
                ],
                userId1: 1,
                userId2: 3,
                city: 'Guangzhou'
            }
        },
        methods: {
            onAction([item, ...rest]) {
                // 请打开控制查看具体事件参数
                console.log(item, rest);
            },
            onChange(value) {
                console.log(value);
            }
        }
    }
</script>

EhComboBoxItemRendererSample.vue文件代码：

<template>
    <div>
        <div v-if="item.id === 1">
            <el-button @click="(e) => {onButtonClick(e, 'action1')}">定义动作1</el-button>
            <el-button @click="(e) => {onButtonClick(e, 'action2')}">自定义动作2</el-button>
            <div>{{item.name}}</div>
        </div>
        <div v-else-if="item.id === 2">Hello, en-ui</div>
        <div v-else @click.stop="stopEvt"><el-slider v-model="item.age"></el-slider></div>
    </div>
</template>

<script>

    import enui from 'ele-enhance-ui';

    const { EhItemRenderer } = enui;

    export default {
        name: 'EhComboBoxItemRendererSample',
        extends: EhItemRenderer,
        methods: {
            onButtonClick(e, action) {
                e.stopPropagation();
                this.dispatchEvent(action)
            },
            stopEvt(e) {
                console.log(e);
            }
        }
    }
</script>
