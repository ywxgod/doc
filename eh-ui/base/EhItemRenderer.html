# EhItemRenderer

# 继承

无

# 描述

用于数据项的渲染，此类定义了基本的属性，方法已经事件。可以用于一般的列表渲染，当需要用于复杂列表渲染时，比如复杂表格项的渲染，我们可以继承此类进行扩展。具体用例可以参考EhSelect。

# 属性

属性
item：Object
某些列表组件，比如EhSelect，EhTable等数据源中的某项数据。

# 方法

方法
dispatch()：void
触发action事件。

# 事件

事件
action：String
当调用dispatch方法时会触发action事件

# 属性详情

item

item为数据源中的某项数据

# 方法详情

dispatch(...rest)

当自定义的itemRenderer需要向其owner组件发送事件时，我们可以调用dispatch方法，dispatch方法默认会发送action事件。在owner组件中我们可以在dispatch方法中带上自定义参数来区分哪个itemRenderer所发出的action事件

# 事件详情

action

当调用dispatch方法时会触发action事件

参数名称 说明
数组：[item, ...rest] item为当前itemRenderer所渲染的options项，来自于options数组。 rest为自定义参数数组，

EhSelectItemRendererSample.vue文件代码：

<template>
    <div>
        <div v-if="item.id === 1">
            <el-button @click="(e) => {onButtonClick(e, 'action1')}">定义动作1</el-button>
            <el-button @click="(e) => {onButtonClick(e, 'action2')}">自定义动作2</el-button>
            <div>{{item.name}}</div>
        </div>
        <div v-else-if="item.id === 2">Hello, en-ui</div>
        <div v-else @click.stop="stopEvt"><el-slider v-model="item.age"></el-slider></div>
    </div>
</template>

<script>

    import enui from 'ele-enhance-ui';

    const { EhItemRenderer } = enui;

    export default {
        name: 'EhComboBoxItemRendererSample',
        extends: EhItemRenderer,
        methods: {
            onButtonClick(e, action) {
                e.stopPropagation();
                this.dispatchEvent(action)
            },
            stopEvt(e) {
                console.log(e);
            }
        }
    }
</script>
